## Запуск проекта

Сборка проекта

```bash
npm run build
```

Запуск проекта в режиме разработки (port:3000)

```bash
npm run dev
```

Запуск node server (port:3000)

```bash
npm run start
```

Запуск тестов

```bash
npm run test
```

Запуск линтеров

```bash
npm run lint
```
